Brief System Description

To offer a convenient solution for a certain assistant/"master" to fill out a new Work Order.

A Work Order is a document that specifies which people need to complete certain types of work today and, as a result, achieve certain metrics (e.g., extract 1 ton of coal).

The user creates an order using the following four types of initial information:

  • Employee list, from which a specific performer is selected;
  • Work type journal, from which a specific type of work is selected for the employee on a given date;
  • Work area;
  • General plan for the month, broken down into daily values.
