BRIEF SYSTEM DESCRIPTION

Real-time tracking and analysis of all data from the mine. Using visualization and analysis tools, there is the possibility of collecting all data together in a customizable enterprise dashboard. This will allow turning raw data on the current state of mining and production into actionable information for management decisions, which can be available to all stakeholders of the enterprise. Ultimately, this will improve decision-making efficiency;

The system also provides the possibility for miners to send a distress signal to the surface in case of any incidents. Strata Commtrac has the capability of online data transmission from sensors (gas, temperature) embedded in the headlamp with location tracking;

Creating any kind of reports. There is the possibility of customizing and creating dynamic graphs and charts, generating any types of reports, such as: overall equipment effectiveness report, downtime monitoring, timesheet accounting, etc. Sending reports within the system to all concerned employees.
